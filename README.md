# 🩻 Chest X-Ray Pneumonia Detection

## 📌 Overview

Deep Learning project for detecting Pneumonia from Chest X-Ray images using CNN and Transfer Learning models.

---

## 🚀 Models Used

- Base CNN
- Advanced CNN
- EfficientNetB0
- ResNet50V2

---

## 📊 Features

- Transfer Learning
- Data Augmentation
- Precision & Recall Metrics
- ROC Curve
- Confusion Matrix
- Model Comparison

---

## 🛠️ Technologies

- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- Matplotlib
- Seaborn

---

## 📈 Results

EfficientNetB0 achieved the best balance between:
- accuracy
- speed
- lightweight deployment
